# GrowthOS - AI原生增长工作台

## 1. Project Description

面向中国出海企业的AI原生增长工作台。通过自然语言与结构化界面，赋能企业在海外市场的研究、获客、内容、运营、销售协同与数据学习全链路。

**核心用户（MVP阶段）：**
- B2B制造与外贸
- 中国SaaS/科技企业
- 专业服务
- 出海营销代理商

**设计原则：** 目标优先、AI辅助、桌面优先、深色科技风主题

**色彩系统：** 深色蓝紫渐变背景 + 科技感光效 + 高级灰阶
- 主色: #6C5CE7 → #A29BFE
- 主背景: #0C0A1A
- 卡片: rgba(44,30,90,0.6) + backdrop-blur


## 6. Development Phase Plan

### Phase 1: 设计系统 + 导航框架 + 今日首页
- Goal: 建立完整的视觉设计系统和导航框架，实现"今日"首页
- Deliverable: 
  - ✅ Tailwind主题配置（颜色、字体）
  - ✅ 全局CSS变量与动态光晕背景
  - ✅ 侧边导航栏 + 顶部栏
  - ✅ 今日首页（卡片式聚合布局）
  - ✅ 全部路由配置与Layout框架
- **新增 — 目标设定向导**
  - ✅ GoalPage（`/goal`）：三步骤引导流程
    - Step 1 目标选择：6 个核心目标卡片（寻找海外客户/招募经销商/推广产品/激活历史客户/运营内容/处理询盘）
    - Step 2 渐进式问答：根据目标动态加载 4-5 个问题，支持单选/多选/文本/下拉/文本域
    - Step 3 结构化计划：AI 生成 4 阶段可编辑计划，确认后进入工作台
