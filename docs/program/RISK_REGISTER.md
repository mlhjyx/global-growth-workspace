# Risk Register（风险登记）

母本 15.9 定义 RISK-001~012（产品/技术基础风险），为权威基线。本文件登记项目执行中新增的风险与缓解跟踪。

## 母本基线（RISK-001~012）

见母本 v2.1 §15.9。要点：R-001 范围过大 / R-002 数据源不合规 / R-003 AI 幻觉 / R-004 误发布外联 / R-005 单模型依赖 / R-006 前端复杂 / R-007 技术债扩散 / R-008 归因不全 / R-009 多租户泄漏 / R-010 媒体成本失控 / R-011 平台 API 变化 / R-012 用户过信推荐。
